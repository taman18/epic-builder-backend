# Express TypeScript Production Skeleton

This project is a production-ready Express.js server using TypeScript, with a recommended folder structure, ESLint, Prettier, Swagger, and environment variable management.

## Features
- Express.js with TypeScript
- Organized folder structure (src, routes, controllers, services, models, middlewares, config, utils)
- ESLint and Prettier for code quality and formatting
- Swagger UI for API documentation
- dotenv for environment variables

## Scripts
- `npm run dev` — Start server in development mode
- `npm run build` — Compile TypeScript
- `npm start` — Start compiled server

## Setup
1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and set your environment variables
3. Run `npm run dev` to start the server

## Folder Structure
```
src/
  config/
  controllers/
  middlewares/
  models/
  routes/
  services/
  utils/
  app.ts
  server.ts
```
